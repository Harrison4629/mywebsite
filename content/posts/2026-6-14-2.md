### 1.以下为一些常用的Apex启动项代码



限制最高帧率为X，若X = 0，则为300帧

```
+fps_max X
```

跳过游戏启动时的开场动画

```
-novid
```

显示游戏内的速度坐标参数

```
+cl_showpos 1
```

将游戏优先级设为高

```
-high
```

设置鼠标原始输入

```
+m_rawinput 1
```



## 2.使用方法

在steam启动选项中填入各个需要的代码，并用空格隔开，如

```
+fps_max 165 -novid +cl_showpos 1 -high
```
