### 第一步：浏览器下载插件

>进入edge浏览器，右上角找到拼图(扩展)图标，并展开点击获取Edge扩展

{{< figure src="/images/2026-6-14-1/1.png" width="20%"  align="center" >}}

{{< figure src="/images/2026-6-14-1/2.png" width="50%"  align="center" >}}

搜索ZeroOmega，并获取它

{{< figure src="/images/2026-6-14-1/3.png" width="80%"  align="center" >}}

### 第二步：配置插件

>获取插件后会自动跳转至Zero Omega管理界面，直接点跳过

{{< figure src="/images/2026-6-14-1/4.png" width="80%"  align="center" >}}

>点击左侧情景模式下的proxy

{{< figure src="/images/2026-6-14-1/6.png" width="40%"  align="center" >}}

>右侧填入配置，协议HTTP，服务器127.0.0.1(即为本地)，端口待定

{{< figure src="/images/2026-6-14-1/7.png" width="80%"  align="center" >}}

#### 这里以代理工具clash verge为例，获取端口

>首先在首页关闭系统代理和虚拟网卡模式

{{< figure src="/images/2026-6-14-1/8.png" width="30%"  align="center" >}}

{{< figure src="/images/2026-6-14-1/9.png" width="30%"  align="center" >}}

>然后在设置中，找到端口(此处端口可自定义)，回到Zero Omega，填入前面待定的端口

{{< figure src="/images/2026-6-14-1/10.png" width="50%"  align="center" >}}

{{< figure src="/images/2026-6-14-1/12.png" width="80%"  align="center" >}}

>最后在左下角点击应用选择

{{< figure src="/images/2026-6-14-1/11.png" width="80%"  align="center" >}}

>然后在第一步扩展图标下点击Zero Omega，并点击勾选proxy

{{< figure src="/images/2026-6-14-1/5.png" width="40%"  align="center" >}}

{{< figure src="/images/2026-6-14-1/13.png" width="40%"  align="center" >}}

### 完工！！！

#### 现在，只有浏览器走代理浏览了！
